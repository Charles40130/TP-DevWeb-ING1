Le HTML


Le html est une lange informatique utilisé sur l'internet.


Il sert à décrire le contenu des pages web.


HTML signifie HyperText Markup Language, c'est à dire langage de balisage d'hypertexte


Comme nous le verrons, ce lanage utilise des balises.


Les balises permettent de mettre en forme le texte, les liens et les images...
